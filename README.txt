How To Use :
1. Jalankan "4 Scheduler.exe" yang terdapat dalam folder Exe
2. Pilih file yang berisi daftar mata kuliah dan persyaratannya (harus dalam format .txt)
3. Pilih DFS untuk menampilkan solusi dengan pendekatan DFS dan BFS untuk menampilkan solusi dalam bentuk BFS
4. Pada window DFS atau BFS, tekan play untuk melihat animasi langkah algoritma DFS/BFS atau pilih prev/next untuk melihat algoritma per tahap.
5. Solusi urutan pemngambilan mata kuliah akan ditampilkan pada text box di bagian kanan window di akhir proses algoritma.

*Source Code berada dalam folder Source > Baca File
